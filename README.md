# Todo List с Локальным Хранением

Это простое консольное приложение для ведения списка задач (Todo List) на Python. Задачи сохраняются в локальном файле `tasks.json`, что позволяет хранить их между запусками программы.

## Функции:

- **Добавление задачи**: Команда `add <текст задачи>` добавляет новую задачу в список.
- **Вывод списка задач**: Команда `list` выводит все задачи с их статусом (выполнена или нет) и временем добавления.
- **Отметить задачу как выполненную**: Команда `done <номер задачи>` отмечает задачу как выполненную.
- **Удаление задачи**: Команда `delete <номер задачи>` удаляет задачу из списка.
- **Выход из программы**: Команда `exit` завершает работу программы.

## Использование

1. Клонируйте репозиторий на вашу локальную машину:

   ```bash
   git clone https://github.com/ваше_имя_пользователя/название_репозитория.git
   
2. Перейдите в директорию проекта:

   ```bash
   cd название_репозитория

3. Убедитесь, что у вас установлен Python (версия 3.x).

4. Запустите программу:
   
   ```bash
   python main.py

5. Используйте команды для работы с задачами.


## Пример команд

#### Добавление задачи

   ```csharp
   > add Купить продукты
   Задача 'Купить продукты' добавлена.
   ```

#### Просмотр списка задач

   ```csharp
   > list
   1. [ ] Купить продукты (добавлена 2024-11-08 00:09:27)
   ```

#### Отметка задачи как выполненной

   ```csharp
   > done 1
   Задача 1 отмечена как выполненная.
   ```

#### Удаление задачи

   ```csharp
   > delete 1
   Задача 'Купить продукты' удалена.
   ```

#### Выход из программы

   ```csharp
   > exit
Выход из программы. До свидания!
   ```

## Файл данных

Все задачи сохраняются в файле tasks.json в формате JSON. Если файл не существует, он будет создан автоматически при первом добавлении задачи, и используется для загрузки задач между запусками программы.

## Дополнительные настройки и расширения

- Если хотите, чтобы задачи сортировались по приоритету или дате, можно изменить соответствующие функции в коде.
- Для удобства использования вы можете добавить уникальные функции, например, поиск по задачам или установку напоминаний.
