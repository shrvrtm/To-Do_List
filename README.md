# Todo List с Локальным Хранением

Это простое консольное приложение для ведения списка задач (Todo List) на Python. Задачи сохраняются в локальном файле `tasks.json`, что позволяет хранить их между запусками программы.

## Функции:

- **Добавление задачи**: Команда `add <текст задачи>` добавляет новую задачу в список.
- **Вывод списка задач**: Команда `list` выводит все задачи с их статусом (выполнена или нет) и временем добавления.
- **Отметить задачу как выполненную**: Команда `done <номер задачи>` отмечает задачу как выполненную.
- **Удаление задачи**: Команда `delete <номер задачи>` удаляет задачу из списка.
- **Выход из программы**: Команда `exit` завершает работу программы.
